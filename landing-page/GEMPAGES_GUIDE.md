## üîß GemPages-Specific Tips

### 1. Using Custom CSS
Add this CSS in GemPages custom CSS section:

```css
/* Smooth hover effects */
.cta-button {
  transition: all 0.3s ease;
}

.cta-button:hover {
  transform: scale(1.05);
  box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.1);
}

/* Gradient backgrounds */
.gradient-red-pink {
  background: linear-gradient(to right, #EF4444, #EC4899);
}

.gradient-orange {
  background: linear-gradient(to right, #F97316, #EA580C);
}

/* Card hover effects */
.feature-card {
  transition: all 0.3s ease;
}

.feature-card:hover {
  box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1);
  border-color: #A855F7;
}
```

### 2. Mobile Optimization
- Test on mobile preview constantly
- Reduce font sizes by 30-40% on mobile
- Stack columns vertically
- Increase touch target sizes (buttons minimum 44px)
- Reduce padding on mobile

### 3. Performance Tips
- Optimize images (WebP format, compressed)
- Lazy load images below the fold
- Minimize custom code
- Use GemPages' built-in elements when possible

### 4. Shopify Integration
- Link CTA buttons to product page or checkout
- Use Shopify's dynamic pricing
- Integrate with cart system
- Add product variants if applicable
